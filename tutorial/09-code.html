Code

Pour créer du code dans une ligne, encadrez le texte avec des accents graves (backticks) `.

Pour créer un bloc de code, vous pouvez soit faire débuter chaque ligne par un retrait de 4 espaces, soit mettre 3 accents graves ``` sur la ligne qui précède le code et celle qui le suit.
